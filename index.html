XR3Player Information

Story

I started developing this application for fun.I had only some months of coding experience so this was a big challenge.The idea is to make a Video,Music,Radio player with extra DJTools and many more...I try to keep the whole think sophisticated especially the UI,so the application is easy to learn and use.
For weekly videos visit my Youtube channel.
